Présentation du Projet SAÉ S1.03

La virtualisation consiste, en informatique, à exécuter sur une machine hôte, dans un environnement isolé, un ou plusieurs systèmes d’exploitation, on parle alors de virtualisation système.
Pour mettre en place un environnement de virtualisation il faut 3 étapes : l’activation de la virtualisation matérielle dans le BIOS, l’installation de l’environnement de virtualisation (VmWare etc), création des machines virtuelles (setup avec l’iso)

Émulateur: Un émulateur reproduit le comportement d'un système matériel ou logiciel sur un autre système. Il imite le fonctionnement d'un matériel spécifique, permettant ainsi d'exécuter des logiciels conçus pour ce matériel sur une plateforme différente.
Exemple : Bluestack est un émulateur permettant d’utiliser ses applications de son smartphone de type APK par exemple sur PC.
Simulateur: Un simulateur reproduit le comportement d'un système ou d'un processus sans nécessairement imiter le matériel sous-jacent.
Outil de virtualisation: Un outil de virtualisation permet de créer des environnements virtuels qui fonctionnent comme des systèmes d'exploitation ou des serveurs distincts sur un même matériel physique.
Exemple : Virtual StudioBox, etc.

Le type de virtualisation pour les environnements comme VirtualBox, VMware Workstation, etc. est le type 2, car il permet d'exécuter une machine virtuelle au-dessus du système d'exploitation déjà installé.
